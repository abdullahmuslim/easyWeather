search help
If you are unable to get your search result with city name, try using coordinates and/or zip/postal code.

Weather can be searched using name, coordinates or zip/post code.

Note:

Coordinates has two value integers representing latitude and longitude of a point on Earth. ie "35 139". They also can have float point.

Zip/Post code is an address use in mailing by different regions of the world. Zip is a two-value input representing zip/post code and the country code.ie "E14 GB". refer to ISO for your country code.

City name can have one, two and three values (three values available in US only) separated with comma Incase place name contains more than a word. ie "London", "London US", "London UK".
